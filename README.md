## ğŸŒŸ About The Project

This is a modern, performance-focused personal portfolio website designed to showcase my skills, projects, and professional background. It is built with a strong emphasis on user experience, smooth animations, and interactive 3D elements.

The design strictly follows a premium dark-mode aesthetic with neon accents (Cyan & Purple) to create a cyberpunk/hacker vibe while maintaining professional elegance.

### âœ¨ Key Features

- **Interactive 3D Background:** A neural network visualization built with React Three Fiber that reacts to mouse movements.
- **Scroll Animations:** Seamless scroll-triggered animations powered by GSAP and Framer Motion.
- **Tech Stack Terminal:** A stylized, type-writing terminal interface that dynamically displays my technical skills.
- **Timeline & Journey:** An elegant vertical timeline component to track my career and education path.
- **Project Showcase:** Interactive 3D tilt-effect cards for displaying project screenshots, tech-tags, and live links.
- **Responsive Design:** 100% mobile-friendly across all devices, ensuring a consistent experience from smartphones to ultrawide monitors.
- **HR & Resume Section:** A dedicated section allowing recruiters to instantly view and download my complete internal resume.

## ğŸš€ Getting Started

If you want to run this project locally, follow these simple steps:

### Prerequisites
Make sure you have Node.js installed (v18.x or later is recommended).

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/umutsibara/my_portfolio_site.git
   ```
2. Navigate into the project directory:
   ```sh
   cd my_portfolio_site
   ```
3. Install dependencies:
   ```sh
   npm install
   # or yarn install / pnpm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
5. Open your browser and visit `http://localhost:3000`.
